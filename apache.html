Tema 3 - Apache

Una Introducción a Apache

Apache es un servidor web ampliamente utilizado que permite alojar y servir páginas web a los usuarios a través de Internet. Es una de las tecnologías más populares para gestionar sitios web, gracias a su flexibilidad y capacidad de soportar múltiples lenguajes y módulos.

El uso de Apache es esencial en configuraciones de servidores web, ya que proporciona funcionalidades avanzadas como redirecciones, manejo de archivos .htaccess y soporte para conexiones seguras con HTTPS.

En entornos modernos, su compatibilidad con otros software y su capacidad de personalización lo hacen una opción confiable para proyectos de cualquier tamaño.

Características y Ventajas de Apache

  • Servidor web HTTP de código abierto
  • Se desarrolla dentro del proyecto HTTP Server(httpd) de la Apache Software Fundation
  • Implementa el protocolo HTTP/1.1
  • Modular
  • Multi-plataforma (Windows, Linux, MacOS)
  • Muy estable
  • Tradicionalmente solución más popular (soporte)

Arquitectura Servidor Apache

Contextos del Servidor

  • Apache organiza su configuración en contextos jerárquicos que se heredan y aplican en diferentes niveles según el alcance deseado. Los principales contextos son:
    • Contexto Servidor
    • Contexto Virtual Host
    • Contexto de Directorio
  • Las configuraciones se heredan: una directiva definida en un nivel superior afectará también a los niveles inferiores, a menos que se sobrescriba en ellos.

Buenas prácticas en directivas

  • DirectoryIndex
    • Establece los nombres de ficheros que servirán como índices al acceder a un directorio sin indicar ningún recurso concreto.
  • Options
    • Controla las características disponibles en un directorio.
  • AllowOverride
    • Permite la configuración de directivas en archivos .htaccess.
  • Order / Require
    • Controla el acceso a los recursos del servidor.

Estructura de carpetas

La estructura de carpetas de Apache está diseñada para organizar de manera eficiente los archivos necesarios para su funcionamiento.

  • Configuración Servidor
    • /etc/apache2
  • Configuración Virtual Host
    • /etc/apache2/sites-available
  • Estructuras de aplicaciones
    • /var/www
